# Expression Evaluator

Proyek ini merupakan sebuah expression evaluator sederhana yang mampu membaca input berupa ekspresi aritmetika, memprosesnya melalui parser, mengubahnya menjadi Abstract Syntax Tree (AST), kemudian mengevaluasi AST tersebut untuk menghasilkan nilai akhir. Alur kerjanya dimulai ketika pengguna mengetikkan ekspresi di REPL. Parser akan memecah string menjadi token, mengenali struktur operasi dan operand, lalu membangun AST yang merepresentasikan ekspresi secara hierarkis. 

Setelah itu evaluator berjalan secara rekursif pada AST untuk menghitung hasil akhir sesuai aturan operator. Kode ini mencakup definisi tipe AST, parser dengan kombinator, evaluator, serta integrasi dengan QuickCheck untuk pengujian otomatis terhadap properti dasar seperti kesesuaian antara parsing dan evaluasi atau sifat-sifat aritmetika tertentu. Sistem ini juga mendukung beberapa operator tambahan seperti modulo, floor, dan ceil, sekaligus menyediakan mekanisme error handling ketika input tidak valid. Dengan struktur ini, proyek menjadi fondasi yang kuat untuk pengembangan lebih lanjut seperti penambahan operator baru hingga perhitungan kalkulus pada minggu berikutnya.
